# GROK OS P2P Node (Прототип)

**GROK OS** — експериментальний P2P-клієнт для децентралізованої ОС із вбудованим ШІ. Прототип симулює обмін ресурсами між вузлами, винагороджуючи токенами за участь.

---

## Особливості

- Використовує до 15% ваших обчислювальних ресурсів (CPU) — лише коли комп’ютер не завантажений.
- Симуляція розподілених обчислень для тестування ідеї (навантаження та заробіток токенів).
- Повністю офлайн, без телеметрії й зовнішніх підключень.
- Працює на Windows/Linux (Python 3.10+).

---

## Встановлення та запуск

1. **Встановіть Python**  
   [Завантажити Python 3.10+](https://www.python.org/downloads/)

2. **Встановіть залежності**  
   Відкрийте термінал та виконайте:
   ```
   pip install psutil
   ```

3. **Запустіть вузол**
   - Збережіть `grok_p2p.py` з цього репозиторію.
   - Запустіть у терміналі/командному рядку:
     ```
     python grok_p2p.py
     ```

---

## Як це працює

- Скрипт моніторить використання CPU.
- Якщо ваш комп’ютер в idle (менше 85% завантаження), бере до 15% ресурсів для "обчислювальної задачі".
- За кожну виконану задачу нараховуються "токени" (GROK) — симуляція майбутньої системи винагород.
